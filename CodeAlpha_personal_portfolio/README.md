## ‚ú® Features

### Core Functionality
- **Responsive Design**: Fully responsive layout that works seamlessly on mobile, tablet, and desktop devices
- **Smooth Navigation**: Fixed navigation bar with smooth scrolling to sections
- **Mobile Menu**: Hamburger menu with animated transitions for mobile devices
- **Scroll Animations**: Fade-in effects triggered on scroll using Intersection Observer API
- **Active Section Highlighting**: Navigation automatically highlights the current section
- **Scroll-to-Top Button**: Convenient button for quick navigation to the top

### Sections
1. **Header/Navbar**: Fixed navigation with smooth scroll links
2. **Hero Section**: Eye-catching introduction with call-to-action buttons
3. **About Me**: Professional introduction and background
4. **Skills**: Technical skills organized in interactive cards
5. **Projects**: Portfolio projects with descriptions and tech stacks
6. **Resume**: Download/view resume options
7. **Contact**: Email and social media links
8. **Footer**: Copyright and credits

## üöÄ How to Run the Project Locally

### Option 1: Direct Opening
1. Download or clone this repository
   ```bash
   git clone https://github.com/yourusername/portfolio-website.git
   ```
2. Navigate to the project folder
3. Open `index.html` in your web browser

### Option 2: Using Live Server (Recommended)
1. Install a local development server:
   - **VS Code**: Install "Live Server" extension
   - **Python**: Run `python -m http.server 8000`
   - **Node.js**: Install `npm install -g live-server`

2. Open the project folder and start the server
3. View at `http://localhost:8000` (or the port specified)
